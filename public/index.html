Shop Book

Introduction

Books shop is a web application which could be used by small shopkeepers to manage customers, products and purchases. Shopkeeper will have to create account using his email and then he could use this shop book to manage his shop.

Features

  • Login
  • Signup
  • Manage Customer
  • Manage Products
  • Manage Purchases

Technologies

  • MongoDB
  • Express
  • React
  • Node JS

Tools

  • VS Code
  • Git & Github
  • Google Chrome
  • Heroku
